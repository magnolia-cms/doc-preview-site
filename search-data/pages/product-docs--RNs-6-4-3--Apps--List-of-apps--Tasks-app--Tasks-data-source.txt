# Tasks data source

The tasks data source provides a list of tasks of a given type via info.magnolia.task.TasksManager. For more information, see [Tasks](../../../../Features/Tasks/).

> **Note:** The tasks data source is part of the Magnolia 6 UI framework. The fully qualified class name is info.magnolia.task.app.TasksDataSourceDefinition.

## [](#_example_definition)Example definition

```yml
datasource:
  class: info.magnolia.task.app.TasksDataSourceDefinition
  sortBy:
    mgnl:lastModified: descending # default order
```

| Property | Description |
| --- | --- |
| `class` | **required** Fully qualified name of the definition class. Use `info.magnolia.task.app.TasksDataSourceDefinition` . |
| `name` | **optional** , *default is tasks* Name of the data source. |
| `sortBy` | **optional** List of properties the data source sorts by. Possible values for [SortDirection](https://vaadin.com/api/framework/8/com/vaadin/shared/data/sort/SortDirection.html) are `ascending` and `descending` . |

| Type | Implementation | Description |
| --- | --- | --- |
| `info.magnolia.ui.datasource.PropertySetFactory` | `info.magnolia.task.app.TasksPropertySetFactory` | Reads and writes properties from/to item in form. |
| `info.magnolia.ui.datasource.ItemResolver` | `info.magnolia.task.app.TasksItemResolver` | Converts path to actual item and back. |
| `info.magnolia.ui.contentapp.ItemDescriber` | `info.magnolia.task.app.TasksItemDescriber` | Displays description for selected item(s) at the bottom of app. |
| `info.magnolia.ui.contentapp.browser.ItemInteractionAvailability` | `info.magnolia.task.app.TaskInteractionAvailability` | Checks whether item is available for interaction (selection, editing, etc). |
| `info.magnolia.ui.observation.DatasourceObservation` | `info.magnolia.task.app.TaskDataSourceObservation` | Observation utility wrapper used in UI framework. |
| `info.magnolia.ui.contentapp.browser.ListPresenter` | `info.magnolia.task.app.TaskListPresenter` | Displays items as list. |
