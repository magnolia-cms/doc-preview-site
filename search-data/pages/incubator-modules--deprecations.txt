# Deprecated incubator modules

This page contains a list of deprecated or discontinued incubator modules.

Deprecated incubator modules are no longer actively maintained or updated. However, the module’s documentation and repository usually remain available for an period of time.

This list is maintained from February 2024.

## [](#_canto_dam_connector)Canto DAM connector

Deprecated: December 2025 Status: Unavailable

The Canto DAM connector module has been deprecated as part of Magnolia’s ongoing initiative to streamline its Incubator module catalog. Over time, the module has remained unchanged and has not gained significant adoption, making it less aligned with our focus on actively maintained solutions.

Existing users are encouraged to explore alternative solutions that align with current best practices.

For more details on the module, see [canto-dam-connector:ROOT:index.adoc](#canto-dam-connector:ROOT:index.adoc).

## [](#_sso_login_extension)SSO Login extension

Deprecated: August 2024 Status: Unsupported

The initial need to enable users to log in with SSO or via a URL using JCR authentication is now met by our [Magnolia SSO module](../../magnolia-sso/) from version 4.0.0. The SSO Login Extension incubator module conflicts with and cannot be installed alongside Magnolia SSO module 4.0.0 and later versions. If you currently use the SSO Login Extension module, please open a [Support ticket](https://magnolia-cms.atlassian.net/secure/CreateIssue.jspa?pid=10188) for assistance migrating to Magnolia SSO 4.0.0.

You should [uninstall](../../sso-login-extension/#_uninstalling) the SSO Login Extension incubator module. We will keep the repository available temporarily to support your migration process.

For more details on the module, see [SSO Login Extension module](../../sso-login-extension/).

## [](#_multi_assets_upload)Multi Assets Upload

Deprecated: February 2024 Status: Unavailable

The Multi Assets Upload module is deprecated since the release of [Magnolia CMS 6.2.43](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.43.adoc). This release of Magnolia introduced asset management enhancements, including bulk asset upload functionality, that supersede the functionality provided by the Multi Assets Upload module.

See [6.2@product-docs:ROOT:Authoring/Managing-assets/uploading-assets.adoc](#6.2@product-docs:ROOT:Authoring/Managing-assets/uploading-assets.adoc) for more details.

For more details on the module, see [Multi Assets Upload module](../../multi-assets-upload/).
