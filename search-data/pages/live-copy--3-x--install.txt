## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](#6.2@product-docs:ROOT:Developing/Bundles-and-webapps.adoc):

```xml
<dependency>
  <groupId>info.magnolia.livecopy</groupId>
  <artifactId>magnolia-livecopy</artifactId>
  <version>3.3.3</version>
</dependency>
```
