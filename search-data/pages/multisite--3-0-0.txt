## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your bundle:

```xml
<dependency>
  <groupId>info.magnolia.multisite</groupId>
  <artifactId>magnolia-module-multisite</artifactId>
  <version>3.0.0</version>
</dependency>
```

```xml
<dependency>
  <groupId>info.magnolia.multisite</groupId>
  <artifactId>magnolia-module-multisite-jcr</artifactId>
  <version>3.0.0</version>
</dependency>
```

## [](#_usage)Usage

For usage, see [How to use Multisite](../../product-docs/6.3/Developing/Templating/Site-definition/How-to-use-Multisite/).

> **Caution:** Since version 3.0.0 of the Multisite module, site definitions are no longer configured in the config folder of this module, but in the sites subfolder of any module.You can configure a site in a YAML definition (preferred) or directly in the JCR Configuration app.
