# Hidden field - 5 UI

Deprecated

> **Note:** This hidden field definition has been deprecated since Magnolia 6.0. It is part of the Magnolia 5 UI framework.For the updated implementation, see Hidden field for Magnolia 6 UI instead.

`HiddenFieldDefinition` defines a hidden field. Magnolia does not render the field in the user interface. You can use it to submit additional parameters in a form such as campaign information. The field does not have any properties of its own.

class: `info.magnolia.ui.form.field.definition.HiddenFieldDefinition`

fieldType: `hidden`

## [](#_hidden_field_properties)Hidden field properties

Simple hidden field definition to store the type of a link:

```yml
form:
  tabs:
    - name: tabLink
      label: Link
      fields:
        - name: link
          fieldType: text
          label: Link
        - name: linkTitle
          fieldType: text
          label: Link title
        - name: linkType
          fieldType: hidden
          defaultValue: external
          type: String
```
