## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../product-docs/RNs-6-4-3/Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.mcp</groupId>
  <artifactId>magnolia-mcp</artifactId>
  <version>1.0-SNAPSHOT</version> (1)
</dependency>
```

|     |     |
| --- | --- |
| **1** | Should you need to specify the module version, do it using `<version>`. |

### [](#_embedded_mcp_server)Embedded MCP server

To enable the embedded MCP server that exposes Magnolia tools and resources:

```xml
<dependency>
  <groupId>info.magnolia.mcp</groupId>
  <artifactId>magnolia-mcp</artifactId>
  <version>1.0-SNAPSHOT</version> (1)
</dependency>
```

|     |     |
| --- | --- |
| **1** | Should you need to specify the module version, do it using `<version>`. |

> **Important:** The embedded MCP server is only available on author instances. It is automatically disabled on public instances, as the exposed tools and resources are intended for content authoring operations only.
