# Mail Tools app

See [Mail](../../../Modules/List-of-modules/Mail-module/Mail/) for more information about using the app.

The Mail Tools app allows you to configure and test mail settings for sending emails with Magnolia. For more information about sending emails in Magnolia see the [Mail module](../../../Modules/List-of-modules/Mail-module/) documentation.

## [](#_configuration)Configuration

The Mail Tools app is installed by the [Mail module](../../../Modules/List-of-modules/Mail-module/). The Tools app group that this app belongs to is only available to the `superuser` role. This is configured in the [app launcher layout](../../../Administration/App-Launcher/App-launcher-layout/). The app is configured in `/modules/mail/apps/mail`.
