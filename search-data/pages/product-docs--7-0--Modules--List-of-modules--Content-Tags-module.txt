## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../../../RNs-6-4-3/Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.contenttags</groupId>
  <artifactId>magnolia-content-tags-ui</artifactId>
  <version>{magnolia-content-tags-64}</version> (1)
</dependency>
```

|     |     |
| --- | --- |
| **1** | Should you need to specify the module version, do it using `<version>`. |

> **Tip:** It is sufficient to add magnolia-content-tags-ui only to the POM file of your webapp and let let Maven handle the dependency to magnolia-content-tags-core.

```xml
<dependency>
  <groupId>info.magnolia.contenttags</groupId>
  <artifactId>magnolia-content-tags-core</artifactId>
  <version>{magnolia-content-tags-64}</version> (1)
</dependency>
```
