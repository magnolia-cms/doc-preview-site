| Latest | 3.0.0 |

The Site module provides a site API. It allows you to create a [site definition](../../../Developing/Templating/Site-definition/) that contains configuration for a whole site such as navigation and common page areas.

## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../../Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.site</groupId>
  <artifactId>magnolia-site-app</artifactId>
  <version>3.0.0</version> (1)
</dependency>
```

|     |     |
| --- | --- |
| **1** | Should you need to specify the module version, do it using `<version>`. |

> **Warning:** The dependency to the site-app is sufficient. Maven will automatically add the site module
