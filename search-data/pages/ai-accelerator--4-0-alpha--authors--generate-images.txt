# Generate images with AI

**Available from version `3.1.0`**

Generate images using AI models directly from the Assets app with a streamlined dialog interface. This feature provides quick access to AI-powered image generation with support for multiple models including DALL-E, FLUX, Gemini, and more.

## [](#_usage)Usage

Access the image generation dialog from the Assets app:

1.  Click **Generate asset** in the action bar.

2.  Select your AI model from the dropdown.

3.  Enter a prompt describing the image you want to generate or use a prompt template.

4.  Configure model-specific parameters (aspect ratio, number of images, image size and so on).

5.  Click **Generate** to create the image(s).

6.  Choose where to save the generated image in the Assets app.
