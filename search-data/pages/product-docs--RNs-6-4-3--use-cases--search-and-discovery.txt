# Search and discovery

![useCase search](../../_images/features/useCase-search.jpg)

For administrators, Magnolia provides a transparent, preconfigured search indexing configuration where you define what content is indexed and how it is weighted. By default Magnolia search functionality is provided by Apacheâ€™s Jackrabbit search.

> **Tip:** For high asset volumes (100,000+ DAM assets) and advanced search requirements, you can configure Magnolia to use Apache Solr instead.

The [Magnolia Travel Demo](../../Getting-started-with-Magnolia/Demo/) includes a complete search example, including how to search for content that is stored in a [content app](../../Apps/Developing-an-app/My-first-content-app/). For front-end developers, Magnolia provides [search templating functions](../../Developing/Templating/Template-scripts/Templating-functions/searchfn/) to render search results using only a template script.
