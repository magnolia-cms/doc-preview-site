# Magnolia Cloud updates 2021

This page lists the updates for Magnolia Cloud in 2021.

## [](#_december)December

### [](#_magnolia_6_2_15)Magnolia 6.2.15

Release notes for [Magnolia CMS 6.2.15](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.15.adoc) are available.

### [](#_magnolia_6_2_14)Magnolia 6.2.14

Release notes for [Magnolia CMS 6.2.14](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.14.adoc) are available.

## [](#_november)November

### [](#_external_dam_1_0_8)External DAM 1.0.8

We improved [S3](https://aws.amazon.com/s3/) caching for bucket regions as there were some performance issues causing problems with the Delivery API.

> **Tip:** See EXTDAM-318 for more details.

### [](#_tomcat_update)Tomcat update

We updated to [Tomcat 9.0.54](https://tomcat.apache.org/tomcat-9.0-doc/changelog.html) due to security issues with [Tomcat 9.0.43](https://tomcat.apache.org/security-9.html).

> **Tip:** See MGNLTOMCAT-20 for more details.

### [](#_magnolia_6_2_13)Magnolia 6.2.13

Release notes for [Magnolia CMS 6.2.13](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.13.adoc) are available.

## [](#_september)September

### [](#_magnolia_6_2_12)Magnolia 6.2.12

Release notes for [Magnolia CMS 6.2.12](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.12.adoc) are available.

## [](#_july)July

### [](#_analytics_1_1_1)Analytics 1.1.1

The latest `dx-core-cloud-webapp` is delivered with the Analytics module.

### [](#_modules_no_longer_recommended_for_magnolia_cloud)Modules no longer recommended for Magnolia Cloud

The following modules are no longer recommended for use in your Magnolia Cloud software bundle due to performance issues:

-   [RSS Aggregator module](../../../product-docs/RNs-6-4-3/Modules/List-of-modules/RSS-Aggregator-module/)

-   [Javascript Models](../../../javascript-models/)


> **Note:** See Cloud offer limitations for more information.

### [](#_magnolia_6_2_11)Magnolia 6.2.11

Release notes for [Magnolia CMS 6.2.11](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.11.adoc) are available.

## [](#_june)June

### [](#_magnolia_6_2_10)Magnolia 6.2.10

Release notes for [Magnolia CMS 6.2.10](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.10.adoc) are available.

### [](#_magnolia_6_2_9)Magnolia 6.2.9

Release notes for [Magnolia CMS 6.2.9](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.9.adoc) are available.

## [](#_may)May

### [](#_relaxedquerychars_property)`relaxedQueryChars` property

The `relaxedQueryChars` property (tomcat configuration) has been adopted to fix issues related to delivery endpoints and special characters.

## [](#_april)April

### [](#_change_in_automatic_backup_time)Change in automatic backup time

The time when your content and configuration is automatically backed up has changed to `03:00 UTC`.

> **Note:** See Backing up and restoring | When is data backed up? for more details.

### [](#_internationalized_domains)Internationalized domains

We now support Internationalized Domain Names (IDNs).

> **Note:** See Clouds domains and certificates | IDNs for more details.

### [](#_magnolia_6_2_8)Magnolia 6.2.8

Release notes for [Magnolia CMS 6.2.8](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.8.adoc) are available.

## [](#_march)March

### [](#_redirect_to_magnolia_author)Redirect to Magnolia Author

You can now [configure redirects to custom domains](../../Managing-environments-using-the-Magnolia-cockpit/Configuring-domain-redirect-rules/#_redirect_to_magnolia_subscription_url) for the Magnolia Author instance.

### [](#_receive_notifications_for_imported_certificates)Receive notifications for imported certificates

When you [import a certificate](../../Onboarding-to-Magnolia-Cloud/Managing-certificates-from-cockpit/#_handle_certificates) in the Cockpit, we now notify you when the certificate is expiring.

### [](#_magnolia_6_2_7)Magnolia 6.2.7

Release notes for [Magnolia CMS 6.2.7](#6.2@product-docs:ROOT:Releases/Release-notes-for-Magnolia-CMS-6.2.7.adoc) are available.

## [](#_february)February

### [](#_import_certificates_via_the_cockpit)Import certificates via the cockpit

We now have a self-service feature on the [Cockpit](../../Cockpit/) that allows you to import your own certificates. This includes re-importing certificates.

> **Note:** See Managing certificates for more details.

### [](#_enhanced_self_service_restart)Enhanced self-service restart

We have enhanced the self-service restart feature.

> **Note:** See Restarting instances for more details.
