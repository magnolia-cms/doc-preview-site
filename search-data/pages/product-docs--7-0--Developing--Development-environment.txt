# Development environment

Developing with Magnolia requires a minimal set of tools.

## [](#_front_end_and_templating)Front-end and templating

-   Text editor for editing [YAML](../YAML/) files and CSS and JavaScript [resources](../Resources/).

-   Application server such as Apache Tomcat, included in the Community and DX Core bundles.

-   [Magnolia CLI](#magnolia-cli:ROOT:index.adoc) is an npm package providing a command line interface to facilitate light development.


## [](#_back_end_and_java)Back-end and Java

-   Java SE Development Kit

-   IDE such as Eclipse or IntelliJ IDEA

-   Application server such as Apache Tomcat

-   Version control system such as [GitLab](https://gitlab.magnolia-platform.com).

-   [Maven](Maven/) to manage project dependencies and build your project.
