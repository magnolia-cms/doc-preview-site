# Text

The Text component is a simple component that allows editors to enter formatted content using a rich text field.

## [](#_preview)Preview

## [](#_fields)Fields

-   `text` (optional) - A rich text field supporting basic formatting.


## [](#_behavior)Behavior

-   If text is provided, it is rendered with minimal default styling.

-   If the field is left empty, the component doesnâ€™t render anything on the page.


## [](#_creating_the_component)Creating the component

To create this component in your project, run:

```console
npm run mgnl -- create-component Text -p text
```

> **Note:** This command adds a local copy of the component to your project. You can fully customize it afterward.
