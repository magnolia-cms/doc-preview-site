# Localized metadata generation tool

The `metadata.generate_localized` tool generates SEO-optimized metadata (title, description, keywords) for a page across all configured locales. It is exposed through the [Magnolia embedded MCP server](../../../magnolia-mcp/developers/mcp-server/) and is available to the AI Accelerator agent.

## [](#_what_it_does)What it does

When invoked, this tool:

1.  Reads the page content at the specified JCR path and serializes it to JSON.

2.  Iterates through all locales configured in Magnoliaâ€™s `I18nContentSupport`.

3.  For each locale, calls an AI model (via the AI Accelerator task executor) to generate metadata.

4.  Saves the generated metadata properties with locale-specific suffixes on the page node (or on a configurable child node).

5.  Returns a summary of the generated metadata to the AI agent.


The tool generates metadata for **all configured locales automatically** in a single invocation. You do not need to call it once per language.

| Property | Value |
| --- | --- |
| **Tool name** | `metadata.generate_localized` |
| **Command catalog** | `metadata` |
| **Command name** | `generateLocalizedMetadata` |
| **Provider ID** | `metadata` |
| **Priority** | `50` |

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `path` | string | Yes | The JCR path of the page to generate metadata for. Example: `/travel/destinations/spain` |
| `repository` | string | No | The JCR repository/workspace to use. Default: `website` |
| `metadataFields` | array (string) | No | List of metadata fields to generate. Default: `['description', 'keywords']` . Supported values: `title` , `description` , `keywords` |
| `metadataNodeName` | string | No | Name of a child node to store metadata in (relative to page node). If not set, metadata is stored directly on the page node. Example: `meta` |

| Parameter | Description |
| --- | --- |
| `pageContent` | The serialized JSON representation of the page content. |
| `templateId` | The Magnolia template ID of the page. |
| `targetLanguage` | The locale for which metadata should be generated. |
| `fieldsToGenerate` | Comma-separated list of metadata fields to produce. |
