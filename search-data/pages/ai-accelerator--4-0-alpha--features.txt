# Features

This page highlights the main features of the AI Accelerator module.

### [](#_conversational_ai)Conversational AI

Smart Chat

Interact with Magnolia through natural language conversations. Get assistance with content tasks, navigation, and automation through a chat interface that understands context and can execute tools.
[Author guide](../authors/ai-chat/) | [Developer guide](../developers/ai-agent/)

### [](#_content_creation_and_structuring)Content creation and structuring

Component generation

Generate page components by selecting a template and providing a topic in a dialog prompt.
[Author guide](../authors/generate-components/) | [Developer guide](../developers/component-generation/)

Text-to-component generation

Transform free-form text into fully populated page components.
[Author guide](../authors/generate-text-to-component/) | [Developer guide](../developers/text-to-component-generation/)

Page generation

Generate full pages using existing template sets with AI assistance.
[Author guide](../authors/generate-pages/) | [Developer guide](../developers/page-generation/)

Story generation

Generate structured story outlines and refine story content.
[Author guide](../authors/generate-stories/) | [Developer guide](../developers/story-generation/)

Content idea generation

Monitor external URLs and generate content ideas through Smart Chat. Content Watch analyzes monitored URLs on-demand or on recurring schedules and creates actionable tasks in Orchestrate.
[Author guide](../../../orchestrate/authors/content-watch/) | [Developer guide](../../../orchestrate/developers/content-watch-config/)

### [](#_personalization_and_optimization)Personalization and optimization

Variant generation

Create personalized content variants for different audience segments.
[Author guide](../authors/generate-variants/) | [Developer guide](../developers/variant-generation/)

SEO metadata generation

Automatically generate SEO and Open Graph metadata, including page titles, descriptions, and keywords.
[Author guide](../authors/generate-seo-metadata/) | [Developer guide](../developers/seo-metadata/)

GEO optimization

Generate AI-optimized structured data for AI search engines and voice assistants, including JSON-LD schema, voice summaries, FAQs, and entity mentions.
[Author guide](../authors/optimize-geo/) | [Developer guide](../developers/geo-optimization/)

Content translation and localized metadata

Translate content and generate localized SEO metadata through Smart Chat. Use MCP tools for instant translation to any configured language and automated metadata generation for all locales.
[MCP tools documentation](../../../content-translation-support-ext/mcp-tools/) | [Usage guide](../../../content-translation-support-ext/mcp-tools/use-mcp-tools/)

### [](#_text_authoring_assistance)Text authoring assistance

Hyper Prompt

Create and reuse advanced prompts with brand tone, SEO optimization, and RAG references to Magnolia content apps.
[Author guide](../authors/use-hyper-prompts/) | [Developer guide](../developers/hyper-prompt/)

Text fields (generated)

Generate text automatically in a text field using a developer-defined prompt with dynamic dialog variables.
[Author guide](../authors/use-ai-text-fields/) | [Developer guide](../developers/text-fields/)

Text fields (assisted)

Edit text using AI tools for summarizing, expanding, changing tone, and other assisted actions.
[Author guide](../authors/use-assisted-text-fields/) | [Developer guide](../developers/assisted-text-fields/)

### [](#_assets_and_media)Assets and media

Image generation

Generate images directly from the Assets app using supported AI models, including DALL·E, FLUX, and Gemini.
[Author guide](../authors/generate-images/) | [Developer guide](../developers/image-generation/)

Image recognition

Automatically tag images using multimodal large language models for context-aware and semantically rich image analysis.
[Author guide](../../../image-recognition/authors/usage/) | [Developer guide](../../../image-recognition/developers/configure_ai-accelerator/)

Asset metadata generation

Generate asset metadata such as titles, captions, descriptions, and subjects in multiple languages.
[Author guide](../authors/generate-asset-metadata/) | [Developer guide](../developers/asset-metadata/)

## [](#_integration_and_automation)Integration and automation

MCP integration

Connect Magnolia’s Smart Chat to Model Context Protocol (MCP) servers using the Magnolia MCP module with the AI Accelerator module so AI agents can call external tools and Magnolia tasks securely from within the editor.
[MCP module documentation](../../../magnolia-mcp/)

Vector Database

Enable semantic search and RAG (Retrieval-Augmented Generation) capabilities by indexing content as vector embeddings. Provides meaning-based content retrieval with permission-aware results and multi-site support, built on PostgreSQL pgvector.
[Vector DB documentation](../developers/vector-db/)
