## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../product-docs/RNs-6-4-3/Developing/Bundles-and-webapps/):

focal-areas

```xml
<dependency>
  <groupId>info.magnolia.dam</groupId>
  <artifactId>magnolia-image-focal-area</artifactId>
  <version>2.0.0</version>
</dependency>
```

focal-point

```xml
<dependency>
  <groupId>info.magnolia.dam</groupId>
  <artifactId>magnolia-image-focal-point</artifactId>
  <version>2.0.0</version>
</dependency>
```

focal-hotspot

```xml
<dependency>
  <groupId>info.magnolia.dam</groupId>
  <artifactId>magnolia-image-focal-hotspot</artifactId>
  <version>2.0.0</version>
</dependency>
```

focal-samples

```xml
<dependency>
  <groupId>info.magnolia.dam</groupId>
  <artifactId>magnolia-image-focal-samples</artifactId>
  <version>2.0.0</version>
</dependency>
```
