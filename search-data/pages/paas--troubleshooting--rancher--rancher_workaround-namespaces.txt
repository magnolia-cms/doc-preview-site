# Namespace doesn’t appear in Rancher

## [](#_symptom)Symptom

The **namespace** doesn’t appear in [Rancher](https://rancher.com/).

## [](#_next_steps)Next Steps

You’ll need to make sure the `namespace` is moved from **Not in a Project** to **Default**.

1.  Expand Cluster (*for newer versions of rancher*).

    > **Note:** If you are using an older version of Rancher, you only need to click on your cluster and then select Project/Namespaces.

2.  Click on **Project/Namespaces**.

    ![namespaces](../../../_images/troubleshooting/namespaces.png)

3.  Move it to **Default**.

    ![move](../../../_images/troubleshooting/move.png)
