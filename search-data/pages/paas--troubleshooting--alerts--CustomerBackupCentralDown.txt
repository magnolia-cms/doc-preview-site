# Backup central is down

## [](#_symptom)Symptom

A `CustomerBackupCentralDown` alert is firing.

> **Note:** CustomerBackupCentralDown alerts are sent to subscribers via email.

| **Expression** | `up{app="backup-central"} == 0` |
| --- | --- |
| **Delay** | `1` hour |
| **Labels** | `team` |
| **Annotations** | • `summary` • `description` • `tenant` • `cluster_id` • `cluster_name` • `pod` |

## [](#_solutions)Solutions

This section provides solutions that should help resolve the issue in most cases.

### [](#_investigate_why_backup_central_is_down)Investigate why Backup central is down

Possible causes for the backup central not running:

-   Metrics are not being collected from backup central: missing ServiceMonitor, misconfigured ServiceMonitor, metrics from cluster are not being collected

-   Backup central is crashlooping / restarting

-   Backup central is not installed
