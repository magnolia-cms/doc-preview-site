# API

Magnolia is an API-based architecture that lets you build a modern, flexible, and extensible tech stack. Whether you are a frontend, backend, or full-stack developer, the range of Magnolia application programming interfaces will always have an answer for even the most demanding headless or coupled CMS and DXP scenarios.

## [](#_rest_apis)REST APIs

Magnolia provides several frontend-oriented APIs to enable other software to get real, raw content directly from Magnolia.

-   [Delivery API](../../../../rest/api/delivery-endpoint/)

-   [SPA templating API](SPA-templating-API/)

-   [Commands API](Commands-API/)

-   Nodes and properties APIs
