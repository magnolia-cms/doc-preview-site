# Release notes for Magnolia CMS 6.3.17

[LTS release](../../../../support/) • Delivered on October 16, 2025 • Changelog: [21001](https://magnolia-cms.atlassian.net/issues/?filter=21001)

**Magnolia 6.3.17** primarily addresses the UI freeze issue in Rich Text fields that affected **Magnolia 6.3.15** and **6.3.16**. The fix is delivered via ticket [MGNLUI-9284](https://magnolia-cms.atlassian.net/browse/MGNLUI-9284).

Other notable enhancements include a new export action for all filtered problems in the Problems subapp and a configuration property to trigger session invalidation after a password change. Additionally, this release brings fixes for variant switching, persona previews, and the Notifications app.
