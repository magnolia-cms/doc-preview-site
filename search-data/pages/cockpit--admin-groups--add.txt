# Add a group

## [](#_general)General

### [](#_name)name

A unique identifier for the group. Use lowercase letters, numbers, and hyphens only. Must start with a letter.

### [](#_users)users

Select users to add to this group. Users in the group inherit the roles assigned to the group.

### [](#_paas_client_api)paas-client-api

Roles that grant access to the Cockpit API. Assign these roles to allow group members to interact with DX Cloud programmatically.

### [](#_rancher)rancher

Roles that grant access to the underlying Kubernetes cluster management. Assign these roles to allow group members to perform infrastructure-level operations.
