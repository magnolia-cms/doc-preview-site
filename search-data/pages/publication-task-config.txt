## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../product-docs/RNs-6-4-3/Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.workflow</groupId>
  <artifactId>magnolia-publication-task-config</artifactId>
  <version>6.2.1</version>
</dependency>
```

| Property | Description |
| --- | --- |
| `scheduleTracking` | **optional** , *defaults to `true`* Enable or disable the module with true/false. |
| `dateTimePattern` | **optional** , *defaults to `MMM dd, yyyy hh:mm aa Z`* Set the format of the date to be displayed in the tooltip. |
