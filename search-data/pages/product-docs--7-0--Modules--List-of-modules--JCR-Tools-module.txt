| Latest | 3.0.0 |

The JCR tools module installs the [JCR Tools app](../../../Apps/List-of-apps/JCR-Tools-app/) that includes subapps to execute JCR queries, create workspace dumps, and export and import JCR content.

## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../../Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.jcrtools</groupId>
  <artifactId>magnolia-jcr-tools</artifactId>
  <version>3.0.0</version> (1)
</dependency>
```

|     |     |
| --- | --- |
| **1** | Should you need to specify the module version, do it using `<version>`. |
