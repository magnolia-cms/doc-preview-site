# DX Cloud documentation

**Latest Helm:** [{helm-chart-version}](https://gitlab.com/mironet/magnolia-helm/-/tags) **Archetype:** {paas-archetype-version} **Cloud parent:** {magnolia-latest-version-allBranches}

DX Cloud is a digital [**Platform as a Service (PaaS)**](https://www.magnolia-cms.com/platform/architecture-and-tech-stack/platform-as-a-service.html) offering that provides:

-   Magnolia CMS optimized for running in multiple clouds.

-   A complete application development environment running in the cloud.

-   Managed services for enterprise-grade security.

-   A pre-defined continuous integration pipeline including automated quality, unit and retention testing.

-   Magnolia optimized deployment blueprints for a continuous delivery pipeline.


DX Cloud gives you the power and flexibility of the Magnolia DX Core platform with the added bonus of handling the automation and deployment side of it all.
