# List field

`ListSelectFieldDefinition` renders a list box where more than one item can be selected. If the number of items exceeds the height of the component, a scrollbar will appear.

> **Note:** The comboBoxField and listSelectField field types donâ€™t support natural ordering. Instead, use the linkField with textInputallowed set to true as the better choice for ordering items alphabetically.For JSON data sources, use the JsonComboBoxFieldDefinition.

`ListSelectFieldDefinition` extends the `AbstractMultiselectFieldDefinition` class.

## [](#_example_definition)Example definition

```yml
listSelect:
  $type: listSelectField
  label: Books
  datasource:
    $type: jcrDatasource
    workspace: books
    describeByProperty: name
    allowedNodeTypes:
      - lib:book
```

> **Tip:** You can preview this example using the field-examples-module. See Field examples module for more information.

| Property | Description |
| --- | --- |
| `rows` | **optional** , *default is 4* Number of rows in the field. When set to `0` , the actual number of rows is determined implicitly by the field. |
