# Deprecations

This page provides a comprehensive summary of deprecations that have been announced by Magnolia in accordance with our [support:ROOT:index.adoc](#support:ROOT:index.adoc).

The deprecations listed on this page have occurred since Magnolia 6.3 was released on September 4, 2024. For details on previous deprecations, see [6.2@product-docs:ROOT:reference/deprecations.adoc](#6.2@product-docs:ROOT:reference/deprecations.adoc).

## [](#_rss_aggregator_module_november_12_2025)RSS Aggregator module (November 12, 2025)

-   Current status: **Deprecated**


The [RSS Aggregator module](../../Modules/List-of-modules/RSS-Aggregator-module/) is deprecated.

## [](#_local_image_recognition_submodule_september_4_2024)Local image recognition submodule (September 4, 2024)

-   Current status: **End of Life**

-   Announced in: [Magnolia 6.3.0 Release notes](../../Releases/Release-notes-for-Magnolia-CMS-6.3/#_local_image_recognition_module_discontinued)


We have discontinued the option to use a local, pre-trained neural network for the [Image Recognition module](../../../../image-recognition/). This feature was experimental and relied on outdated and unmaintained libraries. Amazon Rekognition is now the reference implementation for the Image Recognition module.
