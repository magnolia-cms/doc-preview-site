# Search engine optimization

The purpose of SEO is to enhance the volume and quality of traffic directed to a website via search engine results. Magnolia provides a variety of features to help you create an enterprise-level website that is also search engine optimized.

Suggested resources

-   [Getting Enterprise SEO Right: The Processes and The Technology](https://www.magnolia-cms.com/blog/getting-enterprise-seo-right.html)

-   [Webinar: The SEO Workflow](https://www.magnolia-cms.com/library/webinars/seo-workflow-siteimprove.html#video) (recording)

-   Magnolia Extensions provides a [Search Engine Optimization](https://marketplace.magnolia-cms.com/detail/search-engine-optimization.html) module.
