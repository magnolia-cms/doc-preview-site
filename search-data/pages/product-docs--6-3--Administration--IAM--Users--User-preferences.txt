# Editing user profiles

User can edit their preferences themselves in the **Edit user profile** dialog. System administrators may also [change preferences programatically](../../../../../RNs-6-4-3/Administration/IAM/Users/User-profile-storage/#_changing_user_profiles_programmatically).

1.  Next to your username in the upper-right corner, click the down arrow and select **Edit user profile**.

2.  In the **User profile** tab, change your password, full name and email address.

    ![Edit user profile tab](../../../../../RNs-6-4-3/_images/6-1-users-editUserProfile.png)

3.  In the **Preferences** tab, change your language and [time zone](#_setting_a_time_zone).

    ![The Preferences tab in the Edit user profile section](../../../../../RNs-6-4-3/_images/6-2-Users-editPreferences.png)


> **Note:** SSO module users can only update their preferred language and time zone.

## [](#_setting_a_time_zone)Setting a time zone

Every user can select their own time zone.

1.  Open the [**Edit user profile**](#_editing_user_profile) dialog.

2.  Go to the **Preferences** tab

3.  Set the time zone accordingly.


Magnolia continues to record events such as page creation using the host server time. The recorded time is converted and displayed in the user’s preferred time zone.

![Tooltip displaying the user’s preferred time zone.](../../../../../RNs-6-4-3/_images/6-1-Users-modifiedDate.png)

## [](#_user_login_information)User login information

For peace of mind and added protection, all users can see their latest login information conveniently from the App Launcher. Typically, the login information is found in the top-right of the App Launcher and is displayed in the user’s local time.

Login information format

*Last login: Month DD, YYYY, hh:mm:ss*

![last login information](../../../../../RNs-6-4-3/_images/admincentral/administration/last-login-information.png)

## [](#_adding_and_removing_favorite_apps)Adding and removing favorite apps

You can add and remove apps to and from the favorite apps section. Changes are saved in your profile and aren’t lost when you log out of your account.

1.  Click the **+/-** button in the favorite apps section.

    ![Editing favorite apps](../../../../../RNs-6-4-3/_images/authoring/edit-favourite-apps.png)

2.  Select the apps you want to add or remove.

3.  Click **Save**.
