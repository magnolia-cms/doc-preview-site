# Managing your user profile

To manage your user profile from the Cockpit, select the user icon in the top right corner.

## [](#_accessing_dx_cloud)Accessing DX Cloud

When you sign up to DX Cloud, your credentials grant you access via SSO so that you only need to login once, creating a streamlined user experience whether you are a developer, author, or administrator.

### [](#_log_in_to_dx_cloud)Log in to DX Cloud:

1.  Go to your dedicated DX Cloud URL.

    This is typically `https://cockpit.YOUR_COMPANY.magnolia-platform.com/`. You receive the URL during your onboarding process. If you need assistance, contact [DX Cloud support](https://magnolia-cms.atlassian.net/servicedesk/customer/portal/13).

2.  Enter your credentials.

3.  Start creating amazing content with DX Cloud.

    ![login to paas](../_images/login-to-paas.png)


### [](#_log_out_of_dx_cloud)Log out of DX Cloud

1.  In the top right corner, click the logout icon.


## [](#_user_preferences)User preferences

Set your user preferences in DX Cloud by selecting the user icon .

> **Tip:** Don’t forget to click Update to save your changes.
