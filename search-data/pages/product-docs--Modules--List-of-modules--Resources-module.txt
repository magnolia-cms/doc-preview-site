The Resources module loads [resource files](../../../Developing/Resources/) from all sources - JCR, file system and [classpath](../../../Getting-started-with-Magnolia/Glossary/#_classpath). The module includes the [Resource Files app](../../../Apps/List-of-apps/Resource-Files-app/) to manage your resources.

## [](#_module_structure)Module structure

| ArtifactId | Description |
| --- | --- |
| `magnolia-resources-parent` | Parent reactor |
| `magnolia-resources` | Contains the API and core functionalities, such as info.magnolia.module.resources.ResourcesServlet. |
| `magnolia-resources-app` | Hosts the [Resources Files app](../../../Apps/List-of-apps/Resource-Files-app/). |
| `magnolia-resources-templating` | Contains the [resfn](../../../Developing/Templating/Template-scripts/Templating-functions/resfn/) templating functions. |

## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../../Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.resources</groupId>
  <artifactId>magnolia-resources-app</artifactId>
  <version>5.0.0</version> (1)
</dependency>
```

|     |     |
| --- | --- |
| **1** | Should you need to specify the module version, do it using `<version>`. |

```xml
<dependency>
```
