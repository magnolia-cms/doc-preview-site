## [](#_prerequisites)Prerequisites

-   You should have a Dotdigital account.

-   You must have a [Dotdigital API user](https://developer.dotdigital.com/docs/setting-up-an-api-user). This user and associated credentials ensure authentication for operations and method calls between Magnolia and Dotdigital.


> **Warning:** This module is at the INCUBATOR level.

## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../product-docs/RNs-6-4-3/Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.marketing</groupId>
  <artifactId>dotdigital-integration</artifactId>
  <version>1.0</version>
</dependency>
```

## [](#_configuration)Configuration

After installing the module, you’ll need to make sure the following fields under Dotdigital configuration:

-   `apiPwd` is your Dotdigital API password.

-   `apiUsername` is your Dotdigital API username.

-   `baseApiUrl` is your Dotdigital API base URL.

## [](#_usage)Usage

This section helps you get accustomed with using the Dotdigital Integration module. There are several Dotdigital apps:

-   Address books

-   Contacts

-   Campaigns

-   Segments


![mgnl apps](_images/mgnl-apps.png)

### [](#_connect_magnolia_forms_to_dotdigital)Connect Magnolia forms to Dotdigital

Integrate Magnolia forms with Dotdigital, allowing the contact data submitted in Magnolia to go to Dotdigital.

```yaml
SentContactToAddressBook:
  class: info.magnolia.marketing.dotdigital.form.processors.SendContactToAddressBookProcessor
  addressBookId: 216615 (1)
  enabled: true
```

|     |     |
| --- | --- |
| **1** | The ID of the address book in Dotdigital. |
