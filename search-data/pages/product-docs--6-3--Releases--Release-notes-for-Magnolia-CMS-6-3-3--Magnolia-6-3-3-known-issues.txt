# Magnolia 6.3.3 known issues

## [](#_performance_issues_with_the_assets_app)Performance issues with the Assets app

> **Note:** This issue is described in the Magnolia 6.3.0 known issues. It is addressed in 6.3.4 with MGNLUI-9115.

## [](#_image_thumbnails_not_shown_in_the_asset_chooser)Image thumbnails not shown in the Asset chooser

We’re investigating an issue introduced between 6.3.2 and 6.3.3 where thumbnails are no longer displayed in the Asset chooser dialog.

For more details, see:

-   [MGNLDAM-1622](https://magnolia-cms.atlassian.net/browse/MGNLDAM-1622).


We’re actively working on resolving the issue.
