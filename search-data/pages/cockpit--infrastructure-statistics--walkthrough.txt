# Infrastructure statistics deep dive

## [](#_sidebar_filters)Sidebar filters

The sidebar filters allow you to refine your view by selecting specific criteria.

-   **Date Range**: Select the time period using preset ranges or the calendar picker for custom dates.

-   **Node Name**: Filter statistics by specific node names within your infrastructure.

-   **Memory Unit**: Choose how to display memory sizes (`KB`, `MB`, `GB`, or `TB`).


Click **Apply filters** to update the view.

## [](#_memory)Memory

Displays memory usage and trends for your infrastructure.

For more details on each metric, click **What is this?** for more information at the bottom of each metric.
