# Text field - 5 UI

Deprecated

> **Note:** This text field definition has been deprecated since Magnolia 6.0. It is part of the Magnolia 5 UI framework.For the updated implementation, see Text field for Magnolia 6 UI instead.

`TextFieldDefinition` renders an editable Text field. When the `rows` property is `1` or unspecified, Magnolia renders an HTML `input` element of type `text`. When `rows` is greater than 1, Magnolia renders a `textarea`.

class: `info.magnolia.ui.form.field.definition.TextFieldDefinition`

fieldType: `text`

*![Text field](6.2@product-docs:ROOT:text-field.png)*

## [](#_text_field_properties)Text field properties

Simple text field definition:

```js
form:
  tabs:
    - name: tabText
      label: Text
      fields:
        - name: title
          fieldType: text
          label: Title
          rows: 2
          maxLength: 600
```
