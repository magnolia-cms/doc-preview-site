# Release notes for Magnolia CMS 6.3.18

[LTS release](../../../../support/) • Delivered on November 27, 2025 • Changelog: [21199](https://magnolia-cms.atlassian.net/issues/?filter=21199)

**Magnolia 6.3.18** focuses on enhancing stability and reliability across the platform. It addresses several bugs and introduces improvements to refine existing functionalities. Key updates include a new configurable option in the Community Edition to enforce trailing slashes on generated links and the ability to display dimension information for TIFF images in the Assets app.

#### [](#_configure_magnolia_on_tomcat_behind_a_reverse_proxy)Configure Magnolia (on Tomcat) behind a reverse proxy

The default `RemoteIpValve` configuration is included in Magnolia bundles out of the box. [This page](../../Administration/Deploying/Configuring-Magnolia-behind-a-proxy/) explains how to set up Magnolia with Tomcat behind a proxy such as Nginx or Apache, detailing the use of `X-Forwarded` headers and Tomcat’s `RemoteIpValve` to address common issues.
