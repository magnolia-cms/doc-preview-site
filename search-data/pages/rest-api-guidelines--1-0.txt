# Magnolia REST API Guidelines

This documentation defines Magnolia REST API guidelines based on the [Zalando RESTful API Guidelines](https://opensource.zalando.com/restful-api-guidelines/) and enforced via Zally.

## [](#_goals)Goals

-   Consistent REST APIs across Magnolia products

-   High compatibility with Zally rules

-   Clear documentation of deviations and Magnolia-specific conventions
