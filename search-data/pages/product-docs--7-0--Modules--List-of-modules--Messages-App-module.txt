[Messages app](../../../Apps/List-of-apps/Messages-app/) allows you to send messages to other users via the [Notifications app](../../../Apps/List-of-apps/Notifications-app/).

## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../../Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.messages</groupId>
  <artifactId>magnolia-messages-app</artifactId>
  <version>6.4.0</version> (1)
</dependency>
```

|     |     |
| --- | --- |
| **1** | Should you need to specify the module version, do it using `<version>`. |
