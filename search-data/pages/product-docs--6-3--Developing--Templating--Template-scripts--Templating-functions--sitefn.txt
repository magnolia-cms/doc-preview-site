# sitefn

[FreeMarker development only](../../../Renderers/#_renderers_provided_by_magnolia)

`sitefn` templating functions provide information about the current [site](../../../Site-definition/). You can access the `Site` object and the `Theme` currently applied to the site.

## [](#_get_the_current_site)Get the current site

Gets the current `Site`.

### [](#_method_signature)Method signature

[`Site`](../Templating-function-arguments-and-return-types/#_info.magnolia.module.site.site)`site()`

### [](#_arguments)Arguments

none

### [](#_returns)Returns

Returns the current `Site`.

### [](#_usage)Usage

`sitefn.site()`

 \[#assign mySite=sitefn.site()\]

## [](#_get_the_site_of_the_passed_content)Get the site of the passed content

Gets the `Site` of the passed content.

### [](#_method_signature_2)Method signature

[`Site`](../Templating-function-arguments-and-return-types/#info.magnolia.module.site.site) `site(` [`ContentMap`](../Templating-function-arguments-and-return-types/#_info.magnolia.jcr.util.contentmap)`content)`

### [](#_arguments_2)Arguments

| Argument | Description |
| --- | --- |
| `content` | **required**<br><br>The content map whose site you want to get. |

### [](#_returns_2)Returns

Returns the `Site` of the passed `ContentMap`.

### [](#_usage_2)Usage

`sitefn.site(content)`

 \[#assign currentSite = sitefn.site(content)\]

### [](#_related_functions)Related functions

-   [`Site`](../Templating-function-arguments-and-return-types/#info.magnolia.module.site.site) `site(` [`Node`](../Templating-function-arguments-and-return-types/#_javax.jcr.node)`content)`


## [](#_get_a_sites_theme)Get a site’s theme

### [](#_method_signature_3)Method signature

### [](#_arguments_3)Arguments

| Argument | Description |
| --- | --- |

### [](#_returns_3)Returns

### [](#_usage_3)Usage

 \[#assign currentTheme=sitefn.theme(sitefn.site(content))\]
