# Magnolia Cloud update - June 2017

These changes are available in the latest update of Magnolia in the cloud.

Related topics

-   [Supported modules in the cloud](../../../Supported-modules-in-the-cloud/)


## [](#_magnolia_core)Magnolia CORE

This update of Magnolia in the cloud is built on Magnolia CORE 5.5.3.

## [](#_global_availability_in_several_regions)Global availability in several regions

Isolated platforms are supported in different regions: when you on board to Magnolia you can choose the region you require.

## [](#_change_in_provider)Change in provider

The Magnolia cloud platform now runs on [Amazon Web Services (AWS)](https://aws.amazon.com/).

| Old term | Replaced with |
| --- | --- |
| *Site* | *Package* (or *Subscription package* in some places) |
| *Site release* | *Release* |

## [](#_ui_improvements)UI improvements

A number of minor usability enhancements have been made across the Magnolia cockpit.
