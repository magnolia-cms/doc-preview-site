## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../product-docs/RNs-6-4-3/Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.workflow</groupId>
  <artifactId>magnolia-module-workflow-extended-toolbox</artifactId>
  <version>2.0.4</version>
</dependency>
```

## [](#_usage)Usage

The Workflow Extended module offers extended workflow functionalities:

-   Linked assets publication

-   Multiple pages publication


### [](#_linked_resources_publication)Linked resources publication

The Workflow Extended module adds an extra step to the publication process. After the publication of the main node, the system performs a dependency check for all modified resources. If dependencies are found it triggers an automatic publication of these resources.

#### [](#_configuration)Configuration

In the Configuration app, change the property workflow for `reviewForExtendedPublication`.

![reviewForExtendedPublication](_images/reviewForExtendedPublication.jpg)

Figure 1. `/modules/workflow/commands/workflow/activate/activate@workflow`

* * *

### [](#_multiple_pages_publication)Multiple pages publication

The Workflow Extended module adds the option to select several pages for publication. It can be used in combination with the Linked resources publication.
