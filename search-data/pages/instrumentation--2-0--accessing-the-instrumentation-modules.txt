# Accessing the instrumentation modules

The Instrumentation module and the Instrumentation Exporter module reside on the Magnolia Incubator repository on Magnoliaâ€™s Nexus.

> **Note:** You may need to add the Incubator and Add-ons servers to your Maven settings file to access the instrumentation modules.

Maven settings file

```xml
<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0 http://maven.apache.org/xsd/settings-1.0.0.xsd">
  <servers>
    ...

    <server>
      <id>magnolia.incubator.releases</id> (1)
      <username>user</username>
      <password>password</password>
    </server>
    <server>
      <id>magnolia.incubator.snapshots</id> (1)
      <username>user</username>
      <password>password</password>
    </server>
    <server>
      <id>magnolia.addons.releases</id> (2)
      <username>user</username>
      <password>password</password>
    </server>
    <server>
      <id>magnolia.addons.snapshots</id> (2)
      <username>user</username>
      <password>password</password>
    </server>
  </servers>

  ...
</settings>
```

|     |     |
| --- | --- |
| **1** | Ensure you have both releases and snapshots for incubator modules. |
| **2** | Ensure you have both releases and snapshots for addon modules. |

> **Tip:** See Maven setup for help if needed.
