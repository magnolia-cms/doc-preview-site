# Reports deep dive

## [](#_environment)Environment

Choose your **Date Range** in the Calendar Picker or choose a preset range.

## [](#_type)Type

Filter by report type:

-   Monthly

-   Weekly


Use the search field to find specific reports by reference ID or date range.

## [](#_download_custom_report)Download Custom Report

Select your own desired date range to download a custom report containing all subscription metrics and analysis for the specified time period.

## [](#_reference)Reference

Unique identifier for each generated report.

This reference ID helps you locate specific reports and corresponds to the report’s internal tracking system. Use this ID when contacting support about a particular report.

## [](#_type_2)Type

Lists the report type:

-   **Monthly**: Comprehensive 30-day analysis of all subscription metrics

-   **Weekly**: Focused 7-day summary of performance and usage data


Monthly reports provide deeper insights and trend analysis, while weekly reports offer more frequent monitoring snapshots.

## [](#_from_date)From Date

The start date of the reporting period.

Shows when the data collection period begins for the report. This date determines the earliest metrics included in the report analysis.

## [](#_to_date)To Date

The end date of the reporting period.

Shows when the data collection period ends for the report. This date determines the latest metrics included in the report analysis.

## [](#_created)Created

The date when the report was generated and became available for download.

Reports are automatically generated at the end of each reporting period. The created timestamp shows when the report finished processing and is ready for download.

## [](#_details)Details

Click the stacked lines ☰ to reveal more options such as the list below.

-   **Download**: Download a PDF of your report containing all subscription metrics and analysis for the specified time period.
