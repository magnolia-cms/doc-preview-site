The Analytics integration framework modules provide a framework and extensions to inject your analytics data into the Magnolia UI enabling authors and marketeers to make data-driven decisions.

The overall process to configure a dashboard is:

-   [Connect to a data supplier](../analytics-data-suppliers/), such as Adobe Analytics, Google Analytics, Matomo, a file or another third party analytics data provider.

-   [Configure the data](../analytics-configure-app/) available to your end users in the Analytics app.

-   Use the [Analytics app](../Analytics-app/) to build a dashboard and add charts as Widgets.


The framework delivers out-of-the-box extensions, the Analytics app, and extensive [predefined samples](../analytics-chart-and-dashboard-configuration/) to help you understand charts and dashboards. We recommend you build dashboards and charts using the app.

## [](#_module_structure)Module structure

| artifactID | Description |
| --- | --- |
| `magnolia-analytics-parent` | Parent reactor. |
| `magnolia-analytics-amcharts` | Provides a default implementation to connect to amCharts, which provides JavaScript-based interactive charts and maps programming libraries and tools. |
| `magnolia-analytics-google4-connector` | Provides a default implementation to connect to Google Analytics (GA4) using OAuth2. |
| `magnolia-analytics-matomo-connector` | Provides a default implementation to connect to Matomo. |
| `magnolia-analytics-adobe-connector` | Provides a default implementation to connect to Adobe Analytics. |
| `magnolia-analytics` | Provides the foundation and the API for the Analytics add-on. |
| `magnolia-analytics-ui` | Provides the Analytics app, the `demo` dashboard and predefined chart types that you can reuse. |
| `magnolia-analytics-amcharts-samples` | Provides sample dashboards. If you donâ€™t want to have the sample dashboards available, remove this submodule. |

## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../../../Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.analytics</groupId>
  <artifactId>magnolia-analytics</artifactId>
  <version>3.0.0</version> (1)
</dependency>
```

## [](#_connecting_to_a_data_supplier)Connecting to a data supplier

Once the framework is installed, you must define the data supplier(s) for the charts to be displayed in the Analytics app dashboard. For more information, see [Connecting to a data supplier](../analytics-data-suppliers/).
