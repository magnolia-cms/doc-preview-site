# Definitions data source

The definitions data source provides a tree of registries, definition providers and definition properties.

> **Note:** The definitions data source is part of the Magnolia 6 UI framework. There is no fully qualified class name for it.If you work with the Magnolia 5 UI framework, see Content connector instead.

## [](#_example_definition)Example definition

```yml
datasource:
  name: definitions
```

The data source for the Definitions app takes only an **optional** `name` property.

| Type | Implementation | Description |
| --- | --- | --- |
| `info.magnolia.ui.datasource.ItemResolver` | `info.magnolia.definitions.app.data.DefinitionItemResolver` | Converts path to actual item and back. |
| `info.magnolia.ui.contentapp.ItemDescriber` | `info.magnolia.definitions.app.data.DefinitionItemDescriber` | Displays description for selected item(s) at the bottom of app. |
| `info.magnolia.ui.contentapp.browser.TreePresenter` | `info.magnolia.definitions.app.overview.tree.DefinitionsTreePresenter` | Displays items as tree. |
