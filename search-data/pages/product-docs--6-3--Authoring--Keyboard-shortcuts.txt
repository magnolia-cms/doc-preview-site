# Keyboard shortcuts

## [](#_browsing_items)Browsing items

| Action | Shortcut |
| --- | --- |
| Move down in a list | ↓ or J |
| Move up in a list | ↑ or K |
| Move down by a page in a list | Fn + ↓ (macOS) or Page Down (Windows) |
| Move up by a page in a list | Fn + ↑ (macOS) or Page Up (Windows) |
| Expand item in a list | →   |
| Collapse item in a list | ←   |

## [](#_selecting_items)Selecting items

| Action | Shortcut |
| --- | --- |
| Select/deselect item | Space (macOS) or Ctrl (Windows) + Click |
| Select next item | Shift+J |
| Select previous item | Shift+K |
| Select range of items | Shift + Click |

## [](#_find_bar)Find Bar

To use the Find Bar search field in the Admincentral, press **Tab** **Tab** **Space**. Then, enter the search word or phrase and press **Enter**.

If there are any results displayed, use the ↑ and ↓ arrow keys to move through the results. Press **Enter** to open the highlighted result.

## [](#_editing_items)Editing items

| Action | Shortcut |
| --- | --- |
| Execute default action on selected items (if configured) / save changes | Enter |
| Go to edit mode | Enter |
| Go to next item property | Tab |
| Go to previous item property | Shift + Tab |
| Cancel changes | Esc |
| Execute delete action on selected items (if configured) | Backspace |

## [](#_forms)Forms

| Action | Shortcut |
| --- | --- |
| Go to next field | Tab |
| Go to previous field | Shift + Tab |

## [](#_dialogs)Dialogs

| Action | Shortcut |
| --- | --- |
| Save and close dialog | Enter |
| Cancel and close dialog | Esc |
