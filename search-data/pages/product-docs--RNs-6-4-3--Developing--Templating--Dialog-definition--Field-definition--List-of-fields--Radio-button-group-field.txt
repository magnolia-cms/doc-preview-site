# Radio button group field

`RadioButtonGroupFieldDefinition` renders multiple radio buttons where only one can be selected. It extends the [`AbstractOptionGroupFieldDefinition`](../Option-group-field/) class.

## [](#_example_definition)Example definition

This will let you choose an item corresponding to a node from the configured JCR workspace.

```yml
radioGroup:
  $type: radioButtonGroupField
  label: Books
  datasource:
    $type: jcrDatasource
    workspace: books
    describeByProperty: name
    allowedNodeTypes:
      - lib:book
```

> **Tip:** You can preview this example using the field-examples-module. See Field examples module for more information.

| Property | Description |
| --- | --- |
| `layout` | **optional,** *default is vertical* Specifies the layout orientation of the field. Another possible value is `horizontal` . |
