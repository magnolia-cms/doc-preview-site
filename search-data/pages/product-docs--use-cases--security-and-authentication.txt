# Security and authentication

![useCase security](../../_images/features/useCase-security.jpg)

Magnolia offers security and authentication tools, encompassing a default security app module, Single Sign-On (SSO) module, and extendable connectors for comprehensive protection. The default security app module ensures a solid foundation for user access control, allowing administrators to define permissions and roles.

Our SSO module enhances user convenience and security by enabling seamless authentication across multiple applications. Magnoliaâ€™s extendable connectors further bolster security measures, providing flexibility for integration with various authentication systems. This multifaceted approach ensures that organizations can implement stringent security protocols, safeguarding sensitive data and maintaining a secure digital environment within the Magnolia CMS platform.

## [](#_security_and_authentication_modules)Security and authentication modules

Magnolia modules are shipped to you in different ways depending on your license:

-   **Community Edition (CE)**: Modules are included as part of the free and open-source version.

-   **DX Core**: Modules are included as part of the DX Core enterprise product. These modules are in addition to those in the Community Edition license where bundled.

-   **Special**: Some modules need their own special license as they are not included with CE or DX Core licenses. In some cases, an additional license is required.

    > **Tip:** For more details, see License.
