# Code field - 5 UI

Deprecated

> **Note:** This code field definition has been deprecated since Magnolia 6.0. It is part of the Magnolia 5 UI framework.For the updated implementation, see Code field for Magnolia 6 UI instead.

`CodeFieldDefinition` renders a code editor field.

class: `info.magnolia.ui.form.field.definition.CodeFieldDefinition`

fieldType: `code`

![Code](6.2@product-docs:ROOT:code-field.png)

## [](#_code_field_properties)Code field properties

Simple code field definition.

```js
form:
  tabs:
    - name: tabCode
      label: Code
      fields:
        - name: code
          fieldType: code
          language: freemarker
          label: Enter code
```
