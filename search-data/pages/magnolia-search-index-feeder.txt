## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../product-docs/RNs-6-4-3/Developing/Bundles-and-webapps/):

-   Core

-   Algolia

-   Elasticsearch

-   Solr


```xml
<dependency>
  <groupId>info.magnolia.indexfeeder</groupId>
  <artifactId>magnolia-search-index-feeder-core</artifactId>
  <version>1.0.1</version>
</dependency>
```

```xml
<dependency>
  <groupId>info.magnolia.indexfeeder</groupId>
  <artifactId>magnolia-search-index-feeder-algolia</artifactId>
  <version>1.0.1</version>
</dependency>
```

```xml
<dependency>
  <groupId>info.magnolia.indexfeeder</groupId>
  <artifactId>magnolia-search-index-feeder-elasticsearch</artifactId>
  <version>1.0.1</version>
</dependency>
```

```xml
<dependency>
  <groupId>info.magnolia.indexfeeder</groupId>
  <artifactId>magnolia-search-index-feeder-solr</artifactId>
  <version>1.0.1</version>
</dependency>
```

## [](#_usage)Usage

The Magnolia Search Index Feeder enables you to feed Magnolia-managed content into a search engine via publication or depublication events. There are three search engines integrated and ready to use.

-   [Algolia Search Index feeder](algolia/)

-   [Elasticsearch Index Feeder](elasticsearch/)

-   Solr
