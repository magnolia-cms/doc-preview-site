# Infrastructure distribution deep dive

## [](#_cloud_provider)Cloud provider

The cloud provider used for your infrastructure.

**Format**: The cloud provider and region, such as "AWS (eu-central-1)". For example, "AWS (eu-central-1)" means the infrastructure is hosted on AWS in the eu-central-1 region.

## [](#_nodes)Nodes

The number of nodes in your infrastructure.

## [](#_created)Created

The date and time the infrastructure was created.

**Format**: Month, DD, YYYY including relative time, such as "3 days ago".

## [](#_load_balancer)Load Balancer

The load balancer used for your infrastructure. This is the current endpoint exposed by the load balancer.

## [](#_nodes_table)Nodes table

The table of nodes in your infrastructure, showing the node name, status, and other details.

-   **Name**: The name of the node.

-   **Location**: The location of the node.

-   **Node type**: The type of the node.

-   **Version**: The version of the node.

-   **External IP**: The external IP of the node.

-   **Architecture**: The architecture of the node.

-   **Operating System**: The operating system of the node.

-   **CPU usage**: The CPU usage of the node.

-   **Memory usage**: The memory usage of the node.

-   **Created**: The date and time the node was created.


If you want to see all resources (not just Magnolia), click Show all resources. This then shows all pods in the node.
