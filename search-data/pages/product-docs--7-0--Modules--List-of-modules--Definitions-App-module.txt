[Definitions app](../../../Apps/List-of-apps/Definitions-app/) displays active definitions from all sources (JCR and YAML).

## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../../../Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.definitions</groupId>
  <artifactId>magnolia-definitions-app</artifactId>
  <version>4.0.0</version> (1)
</dependency>
```

|     |     |
| --- | --- |
| **1** | Should you need to specify the module version, do it using `<version>`. |
