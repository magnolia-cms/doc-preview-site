# Git

Magnolia source code is version controlled in GitLab. If you are new to GitLab or to distributed version control systems in general, read [Git version control](https://git-scm.com/) first.

More background and details can be found in the [GitLab Handbook](https://handbook.gitlab.com).

-   [Magnolia GitLab public repositories](https://gitlab.com/magnolia-ce)

-   [Install GitLab](https://docs.gitlab.com/install/)

-   [Videos on GitLab best practices](https://about.gitlab.com/getting-started-videos)
