# Managing plugins

This page describes how you can install, uninstall, and update Magnolia CLI plugins in a project.

> **Note:** Below, FooPlugin (cli-foo-plugin) is just an example plugin name.

## [](#_listing_installed_plugins)Listing installed plugins

To list the plugins installed (registered), run the following npm command.

```console
npm run mgnl
```

## [](#_installing)Installing

### [](#_with_add_plugin)With add-plugin

1.  Install it using the following npm command. It installs the plugin and registers it in `mgnl.config.js`.

    ```console
    npm run mgnl -- add-plugin cli-foo-plugin
    ```

    > **Caution:** Windows users using PowerShell must enclose the two consecutive hyphens (--) in quotes:npm run mgnl "--" ...


### [](#_manually)Manually

1.  Run the following npm command.

    ```console
    npm install cli-foo-plugin
    ```

2.  To make it available for the CLI, modify the `mgnl.config.js` file accordingly.

    -   Before modification

    -   After modification


    `mgnl.config.js`

    ```js
    export default {
      plugins: [],
      logger: {
        filename: "./mgnl.error.log",
        fileLevel: "warn",
        consoleLevel: "debug"
      }
    };
    ```

    `mgnl.config.js`

    ```js
    import FooPlugin from "cli-foo-plugin";

    export default {
      plugins: [
        new FooPlugin()
      ],
      logger: {
        filename: "./mgnl.error.log",
        fileLevel: "warn",
        consoleLevel: "debug"
      }
    };
    ```


## [](#_uninstalling)Uninstalling

To uninstall a plugin:

1.  Delete it from the `mgnl.config.js` file.

    -   Before deletion

    -   After deletion


    `mgnl.config.js`

    ```js
    import FooPlugin from "cli-foo-plugin";

    export default {
      plugins: [
        new FooPlugin()
      ],
      logger: {
        filename: "./mgnl.error.log",
        fileLevel: "warn",
        consoleLevel: "debug"
      }
    };
    ```

    `mgnl.config.js`

    ```js
    export default {
      plugins: [],
      logger: {
        filename: "./mgnl.error.log",
        fileLevel: "warn",
        consoleLevel: "debug"
      }
    };
    ```

2.  Execute the following npm command:

    ```console
    npm uninstall cli-foo-plugin
    ```


## [](#_updating)Updating

To update a plugin, run the following npm command. No modification of the `mgnl.config.js` file is necessary.

```console
npm update cli-foo-plugin
```
