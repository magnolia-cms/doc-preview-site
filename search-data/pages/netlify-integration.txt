## [](#_installing_with_maven)Installing with Maven

Maven is the easiest way to install the module. Add the following to your [bundle](../product-docs/RNs-6-4-3/Developing/Bundles-and-webapps/):

```xml
<dependency>
  <groupId>info.magnolia.cdn</groupId>
  <artifactId>magnolia-netlify-integration</artifactId>
  <version>1.0</version>
</dependency>
```

| Item | Description |
| --- | --- |
| `apiKey` | The Netlify API token. `apiToken: your_netlify_api_token` |

| Version | Notes |
| --- | --- |
| `1.0` | Initial release of the module. |
