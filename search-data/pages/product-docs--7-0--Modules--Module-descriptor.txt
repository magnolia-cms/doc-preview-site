# Module descriptor

A module descriptor is a Magnolia-specific file that identifies and defines the module.

-   For a Magnolia Maven module - the module descriptor is an XML file located at `src/main/resources/META-INF/magnolia/<module-name>.xml`.

-   For a Magnolia light module - the module descriptor is a YAML file located at `$magnolia-resources.dir/<module-name>/module.yaml`.


Please be aware that the YAML-based file does not support all features which are possible with the XML-based module descriptor. For more details please have a closer look at the detail pages.

-   [XML-based module descriptor](XML-based-module-descriptor/)

-   [YAML-based module descriptor](YAML-based-module-descriptor/)
