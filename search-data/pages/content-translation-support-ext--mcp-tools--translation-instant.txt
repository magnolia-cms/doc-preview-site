# Instant Translation Tool

The `translation.instant` tool translates a content node from one language to another using a configured translation provider. It is exposed through the [Magnolia embedded MCP server](../../../magnolia-mcp/developers/mcp-server/) and is available to the AI Accelerator agent.

## [](#_what_it_does)What it does

When invoked, this tool:

1.  Reads the content node at the specified JCR path.

2.  Exports the translatable properties as an XLIFF document.

3.  Sends the XLIFF to the configured translation provider (e.g., ChatGPT, DeepL, Google Translate).

4.  Imports the translated XLIFF back into the target language variant of the node.

5.  Returns the result to the AI agent.


The translation is **synchronous**: the tool waits for the provider to return the translated content before completing.

| Property | Value |
| --- | --- |
| **Tool name** | `translation.instant` |
| **Command catalog** | `translation` |
| **Command name** | `instantTranslation` |
| **Provider ID** | `translation` |
| **Priority** | `50` |

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `path` | string | Yes | Path of the content node to translate. Example: `/travel/destinations/spain` |
| `sourceLang` | string | Yes | Source language ISO code (e.g., `en` , `de` , `fr` ) |
| `targetLang` | string | Yes | Target language ISO code (e.g., `en` , `de` , `fr` ) |
| `repository` | string | No | JCR repository containing the content. Default: `website` |
| `providerName` | string | No | Registered translation provider/config ID. Default: `ChatGPT` |

### [](#_guice_registration)Guice registration

The `TranslationToolsGuiceModule` registers both the provider and executor with the MCP infrastructure:

```java
public class TranslationToolsGuiceModule extends AbstractGuiceComponentConfigurer {

    @Override
    protected void configure() {
        super.configure();
        Multibinder.newSetBinder(binder(), McpToolProvider.class)
            .addBinding().to(TranslationToolProvider.class);
        Multibinder.newSetBinder(binder(), McpToolExecutor.class)
            .addBinding().to(InstantTranslationExecutor.class);
    }
}
```

### [](#_command_configuration)Command configuration

The command is registered via YAML bootstrap:

`config.modules.content-translation-support-ext-instant-command.commands.translation.instantTranslation.yaml`

```yaml
class: info.magnolia.translation.ext.instant.command.InstantTranslationCommand
```
