# Breadcrumbs

The Breadcrumbs component is a navigation component that displays the user’s current location within the site’s hierarchy. It renders links based on the Magnolia page structure and can optionally include a custom title.

## [](#_preview)Preview

## [](#_fields)Fields

-   `title` (optional) - Title for the breadcrumbs component.


## [](#_behavior)Behavior

-   Fetches and displays site hierarchy based on page structure.

-   The current page appears as the last item in the breadcrumb trail and isn’t clickable.


## [](#_creating_the_component)Creating the component

To create this component in your project, run:

```console
npm run mgnl -- create-component Breadcrumbs -p breadcrumbs
```

> **Note:** This command adds a local copy of the component to your project. You can fully customize it afterward.
