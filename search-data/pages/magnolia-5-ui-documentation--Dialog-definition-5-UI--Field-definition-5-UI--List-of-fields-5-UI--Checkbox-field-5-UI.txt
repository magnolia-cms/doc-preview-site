# Checkbox field - 5 UI

Deprecated

> **Note:** This checkbox field definition has been deprecated since Magnolia 6.0. It is part of the Magnolia 5 UI framework.For the updated implementation, see Checkbox field for Magnolia 6 UI instead.

`CheckboxFieldDefinition` renders a single checkbox. It is a simpler alternative to [Option group](../Option-group-field-5-UI/).

class: `info.magnolia.ui.form.field.definition.CheckboxFieldDefinition`

fieldType: `checkbox`

![Example fialog](6.2@product-docs:ROOT:checkbox-field.png)

## [](#_checkbox_field_properties)Checkbox field properties

Simple checkbox field definition:

```yml
form:
  tabs:
    - name: tabtitle
      label: Title
      fields:
        - name: showTitle
          fieldType: checkbox
          label: Show title
          buttonLabel: Select to display title
```
