# Admin Users deep dive

## [](#_add)Add

Cick to add a new user. You’ll activate the user and provide his or her email, first, and last name. If relevant to your subscription, you’ll add them to the appropriate groups.

## [](#_download_users)Download users

Download your user list as a CSV file.

## [](#_email)Email

The user’s email.

## [](#_status)Status

User statuses indicate account state:

-   **Active**: User can log in and access the Cockpit.

-   **Inactive**: User account exists but is currently disabled.

-   **Not verified**: User has not yet verified their email address.

-   **Federated**: User added through [Federated IdP](https://docs.magnolia-cms.com/paas/cockpit/sections/admin/#_federations).


## [](#_first_name)First name

The user’s first name.

## [](#_last_name)Last name

The user’s last name (surname, family name).

## [](#_details)Details

Click to show further available actions including:

-   Editing a user.

-   Deleting a user.

-   Resetting a user credentials.


For more details, see [Manage users](https://docs.magnolia-cms.com/paas/cockpit/sections/admin/#_manage_users).
