# Release notes for Magnolia DX Core 6.4.1

Delivered on December 11, 2025 • Page updated on December 12, 2025 • Changelog: [21763](https://magnolia-cms.atlassian.net/issues/?filter=21763)

**Magnolia 6.4.1** is a maintenance release focused on stability and security. This update introduces configurable trailing slashes for links, removes the `maxMergeDocs` workspace limit, offers lighter full-text search index configurations, and enables stricter security by validating that uploaded file extensions match their detected MIME types.

Other notable fixes include correcting translation failures for specific field decorators, ensuring country-specific locales are stored correctly, and resolving XML import errors for SVG assets.

#### [](#_configure_magnolia_on_tomcat_behind_a_reverse_proxy)Configure Magnolia (on Tomcat) behind a reverse proxy

The default `RemoteIpValve` configuration is included in Magnolia bundles out of the box. [This page](../../Administration/Deploying/Configuring-Magnolia-behind-a-proxy/) explains how to set up Magnolia with Tomcat behind a proxy such as Nginx or Apache, detailing the use of `X-Forwarded` headers and Tomcat’s `RemoteIpValve` to address common issues.
